IP & Patent Support

ISCRIPT facilitates protection of intellectual property generated by its members — including patents, copyrights (software code), and industrial designs. We help innovators structure their ideas, evaluate novelty, and prepare strong, compliant filings.

  • Request IP Support
  • Become a Member

What we support

Patents

  • Patentability screening (novelty, inventive step, applicability)
  • Prior art search guidance (keywords, classifications, search strategy)
  • Drafting support: problem, solution, embodiments, claims structure
  • Filing support workflow coordination (where applicable)

Copyright (Software Code)

  • Guidance on protecting original software code and documentation
  • Best practices for versioning, authorship, and ownership clarity
  • Packaging code artifacts for submission and evidence

Industrial Design

  • Support for design protection (appearance, shape, configuration)
  • Preparing drawings / views and design description
  • Novelty check and documentation readiness

IP-Driven Solution Portfolios

  • Bundling related ideas/patents into market-ready solution portfolios
  • Mapping problem → claims → product features → adoption
  • Technology transfer and licensing readiness support

Section 3(k) guidance (Computer-Related Inventions)

For software- and algorithm-heavy inventions, ISCRIPT helps innovators understand Section 3(k) compliance and improve the framing of the invention. This includes:

  • Identifying whether the contribution is only software/algorithm, or tied to a technical effect.
  • Strengthening invention description through system-level implementation details.
  • Focusing on technical problem/technical solution and measurable technical improvement.
  • Improving claim strategy (system + method + apparatus where relevant).

Note: This guidance is provided for research and drafting support; legal decisions and filing outcomes depend on official examination and applicable rules.

How the IP support process works

Step Activity Output
1 Initial intake: idea summary + novelty indicators + use-case Case ID + initial screening
2 Prior art search guidance and claim mapping Search report + novelty gaps
3 Draft structure: background, problem, solution, embodiments Draft specification outline
4 Claim strategy and refinement iterations Draft claims set
5 Filing readiness: figures, proof-reading, final packaging Submission-ready package
6 Portfolio/ToT readiness (optional) Solution portfolio brief

What to submit (recommended)

To speed up evaluation, please prepare the following when requesting IP support:

  • Problem statement (what pain-point is solved?)
  • Proposed solution (core idea in 5–10 lines)
  • System details (components, workflow, architecture, dataset/tools used)
  • Novelty points (what is new compared to known approaches?)
  • Experimental evidence (results, benchmarks, screenshots, prototypes)
  • Target application (industry sector / deployment scenario)
